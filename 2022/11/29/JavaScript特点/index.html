基本说明

  1. JavaScript 变 能改变 HTML 内容 ，变 能改变 HTML 属性 ，变 能改变 HTML 式 样式 (CSS)， ， 能完成页面的数据验证。
  2. JS 需要运行浏览器来解析执行 JavaScript 代码。
  3. JS 是 Netscape 网景公司的产品，最早取名为 LiveScript，后更名为 JavaScript。和 Java 没
    有关系

JavaScript 特点

  1. JavaScript 是一种解释型的脚本语言， ，C 、C++ 等语言先编译后执行 ，而 而 JavaScript 是在程序的运行过程中逐行进行解释。
  2. JavaScript 是一种基于对象的脚本语言，可以创建对象，也能使用现有的对象(有对象)。
  3. JavaScript 是弱类型的，对变量的数据类型不做严格的要求，变量的数据类型在运行过程可以变化
