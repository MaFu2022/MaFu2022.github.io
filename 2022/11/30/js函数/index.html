函数介绍

函数是由事件驱动的，或者当它被调用时，执行的可重复使用的代码块

函数的定义

方式1 function 关键字来定义函数

方式2 将函数赋给变量

注意细节

  1. JS中函数的重载会覆盖掉上一次的定义

  2. 函数的 arguments 在 隐形参数（作用域在 function 函数内）

    隐形参数: 在 在 function 函数中不需要定义，可以直接用来获取所有参数的变量。

    隐形参数特别像 java的可变参数一样。 public void fun( int … args )

​ js 跟 中的隐形参数跟 java 的可变参数一样。操作类似数组

要求: 写 编写 一个函数 。 用于计算所有参数相加的和并返回, 如果实参不是 number, 就过滤掉
