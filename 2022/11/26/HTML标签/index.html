HTML标签说明

HTML 标签用两个尖括号”<标签名>”括起来

HTML 标签一般是双标签，如<标签名>和</标签名>前一个标签是起始标签, 后一个标签为结束标签

2. 两个标签之间的文本是html元素的内容

3. **某些标签称为单标签,因为它只需单独使用就能完整地表达意思,如<标签名/> **

4. HTML 元素指的是从开始标签到结束标签的所有代码。
