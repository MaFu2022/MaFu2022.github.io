CSS介绍及使用

发表于2022-11-28|更新于2022-11-28|JAVAWeb整合
|阅读量:

为什么使用CSS

  1. CSS 指的是层叠样式表 (Cascading Style Sheets)
  2. 在没有 CSS 之前，我们想要修改 HTML 元素的样式需要为每个 HTML ，属性，费心费力。所以 CSS 就出现了。
  3. 使用 CSS 将 HTML 页面的 内容与样式分离提高 web 开发的工作效率(针对前端开发)
  4. CSS 可以让 html 元素(内容) + 样式(CSS)分离，更好的控制页面

CSS快速入门

CSS语法

  1. CSS ： 语法可以分为两部分： (1)器 选择器 (2)
  2. 声明由属性和值组成，多个声明之间用分号分隔

  1. 最后一条声明可以不加分号(建议加上)
  2. 一般每行只描述一个属性
  3. CSS 注释：/注释内容/, 类似 java
