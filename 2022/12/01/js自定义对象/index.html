js自定义对象

发表于2022-12-01|更新于2022-12-01|JAVAWeb整合
|阅读量:

Object形式

定义

  • var 对象名 = new Object(); // 对象实例（空对象）
  • 对象名.属性名 = 值; // 定义一个属性
  • 对象名.函数名 = function(){} // 定义一个函数

访问

  • 对象名.属性
  • 对象名.函数名();

{ } 形式

定义

var 名 对象名 = {
属性名：值, // 定义属性
属性名：值, // 性 定义属性 , 注意有,号 号
函数名：function(){} // 定义函数
};

访问

  • 对象名. 属性
  • 对象名. 函数名（）;
